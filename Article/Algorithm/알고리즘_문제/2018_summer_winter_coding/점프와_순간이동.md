# ✏️  문제 해결방법

- BFS라고 생각했으나,, 그리디로 뚝딱뚝딱 풀 수 있습니다.
- 기본 아이디어는 순간이동을 사용하면 베터리가 소모되지 않으므로 최대한 순간이동을 사용하면 됩니다.
- 순간이동은 2배씩 이동하므로 최종 도달 위치가 짝수일 때 사용하면 됩니다. 
- 최종 도달 위치가 홀수라면 한칸 베터리를 사용해서 이동하면 됩니다.

# ⭕ 최종 풀이

``` python 
def solution(n): # n : 종료지점
    ans = 0
    while True:
        if n in [1, 2]:
            ans += 1
            return ans
        if n % 2 == 1:
            n -= 1
            ans += 1
        n = n // 2
```
